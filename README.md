# xl-vue-cli

>>基于webpack的vue-cli 简易模板


操作流程:

1. git clone  克隆

2. npm install  下载依赖

3. npm run dev  运行

4. npm run build 打包编译


目录结构
  ```
    ├── src                                        //源目录(输入目录 | 运行文件可具体根据自己需求来配置)
    │   ├── assets                                 //资源放置目录
    │   ├── router                                 // 路由 vue-router
    │   │   ├── index.js                           //路由配置  
    │   ├── views                                  // 视图 
    │   │   ├── login.vue                          //默认页面 可删除修改
    │   │   ├── index.vue                          //默认页面 可删除修改
    │   ├── index.js           
    │   ├── app.vue                               
    |── index.html                                // html 模板 
    ├── package.json                              // 依赖json 以及script 命令等等
    |── webpack.config.js                         //webpack配置文件 (需要配置可添加修改该文件)    
    

``` 

 实现功能
 1. 热跟新服务
 2. 本地服务,以及开发
 3. 编译打包 map可添加
 4. vue模板语法编译以及基本开发实现
 5. vue-router 路由使用
 6. 资源引入(图片这些)
 7. vue的语法功能实现使用
